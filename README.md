# Pokedex API

## Description

This is a simple API to manage a pokedex.

---

## Installation

1. Clone the repository
2. Install dependencies

```
$ npm install
```

3. Run database

```
$ docker-compose up -d
```

---

## Stack used

- NestJS
- MongoDB
